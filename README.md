## Быстрый старт

Для быстрого развертывания оберх систем на одном сервере:

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/fegerV/Supabase
cd <project>

# 2. Подготовьте окружение
cp .env.example .env
# Отредактируйте .env с вашими параметрами

# 3. Запустите Docker Compose
docker-compose up -d

# 4. Следуйте пошаговым инструкциям в документации
```

## Структура документации

- **docs/** - все документы по развертыванию и настройке
